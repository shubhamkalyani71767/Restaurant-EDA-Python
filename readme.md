\# Restaurant Data Analysis (Python - Jupyter)



\## Project Overview



This project performs exploratory data analysis on a restaurant dataset to understand customer behavior, tipping patterns, and billing trends.



\## Tools Used



\* Python

\* Pandas

\* Matplotlib

\* Jupyter Notebook



\## Analysis Performed



\* Data loading and inspection

\* Missing value check

\* Statistical summary

\* Total bill distribution

\* Tip vs total bill analysis

\* Gender-wise comparison

\* Smoker vs non-smoker analysis



\## Files



\* restaurant\_analysis.ipynb → Main notebook

\* restaurant.csv → Dataset



\## How to Run



1\. Download repository

2\. Open notebook in Jupyter

3\. Run all cells
